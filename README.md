# Lein-Labz 🚀

**Squad FIAP — Intensivo 30 Dias de Negócios Digitais**

Squad de 4 estudantes FIAP construindo múltiplos negócios digitais em 30 dias usando IA como acelerador.

## Projetos

| # | Projeto | Modelo | Meta Receita |
|---|---------|--------|-------------|
| 1 | Agência de Automação IA | Serviços | R$8k-30k/mês |
| 2 | WhatsApp SaaS | SaaS Recorrente | R$5k-25k/mês |
| 3 | CyberShield Security | Auditoria + LGPD | R$3k-12k/mês |
| 4 | Produtos Digitais | Templates/Packs | R$1k-8k/mês |
| 5 | Conteúdo Monetizado | YouTube/Blog/Cursos | R$500-5k/mês |
| 6 | Consultoria LGPD | Consultoria | R$2k-10k/mês |
